Accessibility Features

  • ARIA Attributes: All tooltips use role="tooltip" and aria-describedby
  • Keyboard Navigation: All elements are focusable with Tab key
  • Focus Indicators: Clear visual focus states with accent color
  • Touch Support: Tap to show/hide with 3-second auto-hide
  • No Auto-Dismiss: Tooltips stay visible while focused/hovered
  • Screen Reader Compatible: Properly announced by assistive technologies
